GitHub に自治会のウェブサイトの雛形 https://imada-test.github.io/を作成してみました。

GitHub を利用する方法には、いろいろな利点があるように思います。

● GitHub Pages とは

GitHub Pages を使用して、自分自身、organization、プロジェクトに関する Web サイトを、GitHub のリポジトリから直接ホストできます。

https://docs.github.com/ja/pages/getting-started-with-github-pages/what-is-github-pages

● GitHub Pages ドキュメント

GitHub Pages は、 GitHub リポジトリをライブ Web サイトに変換します。個別のホスティングは必要ありません。

https://docs.github.com/ja/pages
